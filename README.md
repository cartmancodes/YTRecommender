# YTscraper
Script to fetch relevant quality Youtube Videos.

# Sample Command:
CMD:
python YTlib/get_content.py -c 'chill indie playlist' -p 4
python YTlib/get_content.py --content 'chill indie playlist' --pages 4

where c/content: Search Text, p/pages: number of pages to scrape

# Quality paremeters:
Minimum views:10K, Like to Dislike ratio > 100
